Computer Science and Statistics

As a Computer Science student I've completed courses introducing Java, and Python but I've also completed personal projects in C, Python, HTML, CSS, Bash and Lua. I have a strong passion for all types of technology. I especially enjoy using my programming and design skills to personalize that technology.

A good example of this is a project I did in my final year of highschool. Many of my friends had purchased RGB LED light strips for their rooms and I wanted too as well. But, I also wanted to take it to the next level and use individually addressable strips instead of only being able to display one color at a time. I spent a few weeks learning C and with the help of an Arduino I created a controller for that strip with a desktop GUI written in Python. It was lots of fun playing with all the different color combinations and I learned a lot about lower level languages and inter-device communication. Looking back I see a lot that could be improved but considering I started the project without ever touching C I am very happy with the results.
